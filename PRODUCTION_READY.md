## Current Implementation

```typescript
// src/pages/Upload.tsx
const handleAnalyze = async () => {
  // Real API call only - no mock data
  const response = await analyzeAura(selectedFile);
```

### AI Pipeline (Production)

1. **User uploads image** → Upload.tsx
2. **Image converted to Base64** → fireworks.ts
3. **Phase 1: Qwen 2.5 VL 72B analyzes image** → Real API call
4. **Phase 2: LLaMA 3.3 70B generates roast** → Real API call
5. **Results displayed** → Results.tsx

## Monitoring & Debugging

### Check API Calls

Open browser DevTools → Network tab:
- Look for calls to `api.fireworks.ai`
- Check request/response payloads
- Verify status codes (200 = success)

### Check Console Logs

The app logs:
- API errors
- JSON parsing issues
- Network failures

### Common Issues

**Issue:** "No content received from Qwen Vision"
- **Solution:** Check API key is valid and has credits

**Issue:** "Failed to extract JSON"
- **Solution:** Model response format issue, check Fireworks.ai status

**Issue:** Long processing times (>60s)
- **Solution:** Check network connection, try smaller image
